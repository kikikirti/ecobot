# Economics Explainer Bot (CLI) — Transformers + Free HF API

A simple exam-friendly Economics chatbot that explains concepts with structure:
Definition → Intuition → Example → Common mistakes → 3 MCQs.

## Setup

1) Create a Hugging Face token:
- https://huggingface.co/settings/tokens

2) Create `.env` from `.env.example`:
- Copy `.env.example` to `.env`
- Put your `HF_TOKEN`

3) Install dependencies:
```bash
pip install -r requirements.txt
```
4) Run:
```bash

python -m app.main
```
